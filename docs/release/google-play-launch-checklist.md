# Google Play Launch Checklist

## Store content

- App title
- Short description
- Full description
- App icon
- Feature graphic
- Phone screenshots
- Tablet screenshots if required by the target device catalog
- Privacy policy URL

## Compliance

- Data safety form
- Content rating questionnaire
- App access instructions if the game ever adds authentication
- Ads declaration if monetization is added later
- Country and device availability review

## Release readiness

- Internal testers installed and validated the current build
- Closed-track rollout approved after internal QA
- Release notes prepared from the workflow artifact
- Support contact email and website confirmed
